Git

git 환경

git status
git 상태보기

Git 세팅

git init
지금부터 이 폴더에서 작업을 시작하겠다.
git add [파일명][.]
이 파일들을 버전 관리 하겠다.[tracked]
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
사용자가 누구인지 기록한다.

삭제

git reset
스테이지 Area에서 working stage로 back

복사하기

git clone [url]
원격 저장소에 있는 파일을 복제한다. (한번 확인을 하고 작업하고 싶다면 clone)
git pull
원격에 있는 파일의 내용과 로컬에 있는 내용이 자동으로 추가된다. (신뢰성이 있다면 pull로 당겨올때 사용)
git fetch [url]
원격에 있는 자료를 가져옴

복사 만들기

git branch
새로운 브런치로 폴더생성
git checkout [브런치명]
해당 브런치 폴더로 들어감
git switch 브런치명 or master
해당 브런치 폴더로 들어감
